About PrivShare

PrivShare is a cybersecurity lab project that demonstrates anonymous file sharing with two privacy layers.

  1. Metadata cleaning: hidden metadata is removed before sharing.
  2. Password-based encryption: the cleaned file is encrypted before storage.

No user account is created. The receiver needs both the unique link and the password.
