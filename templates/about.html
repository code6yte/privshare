About PrivShare

PrivShare is a privacy-focused file sharing tool. It strips hidden metadata from files, encrypts them with a password you choose, and generates a unique share link. The receiver needs both the link and the password to decrypt.

Upload → Clean metadata → Encrypt → Share link

How it works

Files can leak private information through hidden metadata — GPS coordinates, device identifiers, author names, and editing history. PrivShare removes this metadata before encryption.

The cleaned file is then encrypted using AES-256-GCM with a key derived from your password via PBKDF2-HMAC-SHA256. The password is never stored on the server or placed in the URL.

Stack

Flask backend, AES-256-GCM encryption, ExifTool for metadata cleaning, Pillow for images, pypdf for PDFs, and SQLite for file records.
